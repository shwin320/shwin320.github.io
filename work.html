Late Bus

Don't no which late bus you're on? Not a problem anymore with Late Bus. Just press the "Enter Address" button when you're home, and the address will be saved in the app for future use. Use the map to look up which late bus you take. Map not clear enough? No problem! Just search the official late bus list to see which late bus you take in your area. Never be clueless with what late bus you take again, with Late Bus. (Installation instructions here; intended for use only within Murray Avenue School)

Pallet

Pallet is a drawing pad based on the JavaScript coding language. You can use Pallet to draw and doodle. You can change the background color and draw with 19 color options and 4 brush shapes. It's also easy to remix the code to create new functions and expand Pallet's abilities. All the controls are from the keyboard; the only thing you need your mouse for is to draw with.
